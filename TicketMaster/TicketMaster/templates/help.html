Functionality Information

Viewing Tickets

After logging in, the first screen has four tabs, one with all open tickets, in-progress tickets, closed tickets and all with the open tickets tab selected by default. The tabs contain a list of all tickets and to see more detail, you can click on one of the ticket entries in the list to open a dialog box for that ticket for a full description of the ticket. Closing the dialog box closes the detail view for that ticket.

Creating a New Ticket

After logging in, you click the “New Ticket” button. A dialog box pops up and you can fill in the following fields:

  • Location (the building and room the issue is located in)
  • Requestor (the person who originally reported the issue)
  • Recipient (the person who will benefit from the fix, often the same as the requestor)
  • Urgency (select one of several urgency levels)
  • Status (could be “investigating,” “awaiting replacement equipment,” “resolved” when closed, etc.)
  • Description (a description of the issue as originally reported)

Then you can exit out of the dialog box by clicking the “Create” button to finish creating the ticket, or “Cancel” to abort creating the ticket.

Updating a Ticket

You can open a ticket in the detailed view and there will be a button to accept a ticket that can be clicked. You will not be able to edit the original ticket description message, but can add a follow-up message that will be appended with a new time stamp to the original description for which will be added to the update log. You will also be able to change the urgency level and status of the ticket. Then you can click “update” or “cancel”.

Closing a Ticket

When updating a ticket, there will be a “resolve” button in the dialog box.You can click the button to which you will be prompted to describe how the issue was resolved. Then you can click resolved, and the ticket status is automatically set to "resolved" and the ticket is offically closed. This will move the ticket from its respective tab into the list of closed tickets.

Helpful Tips

When hovering over the description of a ticket, you will be able to see a pop up box that will give you the full description of a given ticket.
